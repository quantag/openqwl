OpenQWL

Open Quantum Workflow Language

A vendor-neutral YAML language for describing complete quantum computing workflows.

OpenQWL describes:

- data import
- preprocessing
- problem formulation
- circuit generation
- execution
- benchmarking
- reporting

OpenQWL is independent of any SDK or hardware vendor.
